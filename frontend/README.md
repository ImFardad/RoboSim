# RoboSim Frontend

رابط کاربری پلتفرم RoboSim — ساخته‌شده با Vue 3، Vite و TypeScript.

## اجرا

```bash
# نصب وابستگی‌ها
npm install

# حالت توسعه
npm run dev      # http://localhost:5173

# build برای تولید
npm run build

# پیش‌نمایش نسخه‌ی build‌شده
npm run preview
```

## نکات

- **فونت:** از فونت وزیرمتن (Vazirmatn) استفاده می‌شود که به‌صورت کاملاً محلی (بدون CDN) نصب شده است.
- **راست‌چین (RTL):** رابط کاربری راست‌چین و فارسی است.
- **اتصال به سرور:** فرانت‌اند به‌طور پیش‌فرض به `http://localhost:3000` وصل می‌شود (backend).
