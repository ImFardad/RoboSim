# 🏗 معماری RoboSim

این فایل مرجع اصلی طراحی و معماری پروژه است و در طول توسعه به‌روزرسانی می‌شود.

---

## نمای کلی

RoboSim یک پلتفرم مسابقه ربات‌های برنامه‌نویسی‌شونده است که از سه بخش اصلی تشکیل می‌شود:

1. **موتور شبیه‌سازی (Simulation Engine)** — هسته‌ی منطقی بازی، مستقل از رابط کاربری
2. **Backend** — مدیریت ربات‌ها، کاربران، مسابقات و API
3. **Frontend** — نمایش گرافیکی میدان نبرد، کنترل مسابقه و ادیتور کد

---

## تصمیمات کلیدی طراحی

| موضوع        | تصمیم                                      |
| ------------ | ------------------------------------------ |
| سبک محیط     | دو‌بعدی از بالا (Top-down 2D)              |
| مدل زمانی    | تیک‌محور گسسته (Tick-based)                 |
| زبان ربات‌ها | TypeScript (همان زبان موتور)               |
| مکانیک بازی  | سبک Robocode (حرکت، اسلحه، رادار، HP/انرژی) |
| فرمت مسابقه  | انعطاف‌پذیر ۱ تا N ربات                    |
| قطعیت        | موتور کاملاً Deterministic (برای Replay)   |

> جزئیات بیشتر در طول پیاده‌سازی به این فایل اضافه خواهد شد.
